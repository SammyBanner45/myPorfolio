# 🌌 Personal Portfolio

A modern developer portfolio built with **React**, **Tailwind CSS**, and **GSAP**, featuring a futuristic neon-inspired design, animated hero section, interactive project showcase, and dynamic tech stack display.

## ✨ Features

* Animated Hero Section with glowing rings and particles
* Smooth scrolling navigation
* Interactive Tech Stack section
* Project Carousel with custom animations
* Neon-inspired UI with cyan and gold accents
* GSAP-powered transitions and motion effects
* Contact Section with social links

## 🛠️ Tech Stack

* React
* Tailwind CSS
* JavaScript
* GSAP
* React Icons

## 📂 Sections

### Hero

An animated landing section featuring glowing rings, particles, and smooth entrance animations.

### Intro

A brief introduction highlighting who I am and what I build.

### Tech Stack

A visual showcase of technologies and tools I use.

### Projects

A curated collection of projects with live demos and source code links.

### Contact

Quick access to my professional profiles and contact information.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/SammyBanner45/myPorfolio.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
